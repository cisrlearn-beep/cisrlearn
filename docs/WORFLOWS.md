- Flow Diagram:  
  ![Certificate Generation](link-to-certificate-generation-diagram)
- Process steps:

  3. Certificate sent via email.

  ![Payment Processing](link-to-payment-processing-diagram)
- Process steps:

  2. Payment gateway processes the transaction.
